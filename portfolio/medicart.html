Medicart, E-commerce website for medicines

Description

Medicart is an E-commerce solution developed for Motherson INfotech & Desgn Systems (Mind). It is developed for selling medicines and related products online. It also contains an Admin Dashboard using which each and every aspect of the website can be controlled, from products to banners and advertisments.

Features

  • Highly responsive design suitable for mobile browsers.
  • Fully adjustable store frontend.
  • Various type of reports like orders, customers, sales etc, generated for store owner.
  • Functionality to search products, product comparison, wishlist, full user account etc.
  • Step by Step Checkout process.
  • Chart representation of data.

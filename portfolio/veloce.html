Veloce - PHP Search Engine

Description

Veloce has been developed using PHP/Mysql. One can control the full functionality of search engine using voice navigation and hand gestures.

Features

  • Follows server side redirections.
  • Allows crawling to be limited by depth (i.e. maximum number of clicks from the starting page), by (sub) domain or by directory.
  • Keywords can be entered by voice (for Goole Chrome Browser)
  • Possibility to exclude common words from being indexed.
  • "Did you mean" search suggestion on mistyped queries.
  • Context-sensitive auto-completion on search terms (like Google Suggest).
  • Word stemming for english (searching for "run" finds "running", "runs" etc).
  • Admin panel for administering the indexed sites.
  • Functionality of Locating addresses on maps.

Technology

  • HTML 5
  • CSS 3
  • Mysql
  • PHP
  • Jquery
  • Javascript
  • Ajax
